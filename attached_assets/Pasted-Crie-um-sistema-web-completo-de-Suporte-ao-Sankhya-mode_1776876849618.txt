Crie um sistema web completo de Suporte ao Sankhya, moderno, responsivo e de fácil manutenção, seguindo rigorosamente as especificações abaixo.

Objetivo do Sistema

Desenvolver uma plataforma interna para suporte operacional e técnico relacionado ao ERP Sankhya, com foco em:

Controle de chamados internos
Consulta rápida de dados espelhados do banco Sankhya
Acompanhamento de liberações
Cadastro de clientes/usuários
Relatórios operacionais
Dashboard com indicadores em tempo real
Interface simples, rápida e intuitiva
